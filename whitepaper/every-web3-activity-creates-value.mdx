# Every Web3 Activity Creates Value

Web3 is made of activity.

Depositing,  
voting,  
joining AI projects,  
using new chains,  
holding NFTs,  
and growing communities all move ecosystems forward.

But most activity fades over time.

RocX is built on the belief that meaningful activity should become lasting value.

<Info>
Meaningful activity should never disappear.

## Activity is Web3's Greatest Asset

Today's Web3 grows through many projects, networks, and communities moving together.

Yet user activity often remains locked inside each project where it happened.

Some users grow communities for years. Some hold NFTs through cycles. Others keep using protocols across different ecosystems.

These actions matter, but they have rarely been recognized as shared value.
