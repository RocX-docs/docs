# The Proof of Activity Market

## A New Infrastructure for Web3

Web3 is creating more activity every year.

But there is still no shared infrastructure for recognizing that activity through common standards and connecting it to financial value.

RocX defines this new market as the Proof of Activity Market.

<Info>
The next generation of Web3  
won't be built  
only on ownership.  
It will be built

</Info>

---

## Web3 creates more activity.

New chains, new protocols, new applications, and new financial services keep emerging.

User activity is increasing, but that activity remains fragmented inside each project.

---

## Infrastructure for connecting activity is still missing.

Today, Web3 shares ownership.

It also shares transactions.

But it does not share activity.

When users move to a new project, their activity starts over again.

Proof of Activity is a new layer that connects this gap.

---

## A new category

RocX sees PoA not as another product, but as new infrastructure.

Proof of Activity is a shared layer that connects diverse Web3 activity into one value system and links it to financial services and new opportunities.

---

## Why Now?

Web3 is no longer only a token market.

DeFi, AI, RWA, L1, L2, BTCFi, ZK, and many other ecosystems are growing together.

What is needed now is not another project, but shared infrastructure that connects these activities.
