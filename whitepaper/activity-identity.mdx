# Activity Identity

Activity Identity defines users through activity history and trust, not only through wallet addresses. RocX connects user context across Web3 to financial experience.

## Beyond Wallet Address

A wallet address can identify a user, but it does not explain their activity or trust. RocX uses activity history to create a richer user profile.

## What Builds Activity Identity

NFT/PFP verification, Web3 activity, DeFi activity, and mission history can all contribute to Activity Identity. The important point is that activity must become a verifiable and meaningful signal.

## Your Activity Defines You

The RocX message is clear: **Your Activity Defines You.** User activity becomes part of identity that can connect to DeFi Planet benefits.
