# Value Flywheel

RocX is designed as a loop where activity creates value and value encourages more activity.

The system grows when users continue participating.

---

## The Loop

Users act, verify, accumulate Active Energy, receive benefits, and return for more activity.

This creates a repeating product experience across the ecosystem.

---

## Activity to AE

Activity becomes useful when it is connected to Active Energy.

AE turns participation into a value layer that can move through RocX.

---

## AE to DeFi Benefits

Active Energy can support better benefits and opportunities inside DeFi Planet.

This makes activity visible inside the financial experience.

---

## Retention and More Activity

Benefits can encourage users to return.

When users return, they create more activity and strengthen the loop.

---

## RocX Growth Direction

RocX grows by connecting user activity, financial experience, and ecosystem participation.

The flywheel turns participation into lasting product value.
