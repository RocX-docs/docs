## 탈중앙화 금융(DeFi)이 부족한 것은 무엇일까요?

탈중앙화 금융(DeFi)은 눈부신 성장을 거듭해 왔습니다. TVL(총 예치 자산)은 수십억 달러에 달합니다. 매일 새로운 프로토콜이 등장하고 있으며, 금융 상품은 점점 더 정교해지고 있습니다.

하지만 성장만으로는 지속가능성을 보장할 수 없습니다. 대부분의 프로토콜은 유동성 확보를 위해 경쟁하지만, 참여를 위해 경쟁하는 프로토콜은 드뭅니다.

결과적으로, 사용자들은 인센티브를 쫓고, 이 프로토콜에서 저 프로토콜로 이동하며, 장기적인 관계를 구축하는 경우는 드뭅니다.

기존 DeFi는 다음과 같은 지표를 측정합니다. 하지만 다음과 같은 질문은 거의 하지 않습니다.

| 기존 DeFi가 측정하는 것 | 기존 DeFi가 거의 묻지 않는 것 |
| --- | --- |
| 예치금 | 얼마나 오래 이용했는가? |
| 대출금 | 얼마나 기여했는가? |
| 유동성 제공량 | 얼마나 활동적인가? |
| — | 얼마나 신뢰할 수 있는가? |

활동성은 가치가 있지만, 활동성은 눈에 보이지 않습니다.

평판은 의미가 있지만, 평판은 단편적입니다.

이러한 단절은 근본적인 한계를 만들어냅니다.

</Warning>
