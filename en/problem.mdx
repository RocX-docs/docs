## What is DeFi missing?

DeFi has grown remarkably. TVL reaches billions of dollars. New protocols launch every day, and financial products keep growing more sophisticated.

But growth alone cannot guarantee sustainability. Most protocols compete for liquidity, but few compete for participation.

As a result, users chase incentives, move from one protocol to another, and rarely build long-term relationships.

DeFi measures the following metrics. But it rarely asks the following questions.

| What DeFi measures | What DeFi rarely asks |
| --- | --- |
| Deposits | How long have you stayed? |
| Borrowings | How much have you contributed? |
| Liquidity provided | How active are you? |
|  | How trustworthy are you? |

Activity has value, but activity is invisible.

Reputation has meaning, but reputation is fragmented.

This disconnect creates a fundamental limit.

<Warning>
A financial system that measures only capital cannot fully understand a person.
